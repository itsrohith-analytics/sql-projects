# Quiz Result Tracker Database (SQL)

## Project Overview
This project represents a Quiz Result Tracker Database designed to store and analyze student quiz performance.  
It helps track quiz scores, student performance, and quiz-wise results using SQL queries.

---

## Database Entities
- **Student** – Stores student information
- **Quiz** – Stores quiz details
- **Result** – Stores quiz scores for students

---

## Key Business Questions Answered
- List of all students and quizzes
- Student-wise quiz scores
- Quiz-wise average scores
- Identification of high-performing students
- Quizzes with low average performance
- Students scoring above average

---

## SQL Concepts Used
- Relational database design
- Primary and foreign keys
- INNER JOIN
- Aggregate functions (AVG, COUNT)
- GROUP BY and HAVING clauses
- Subqueries

---

## Files Included
- `schema.sql` – Database structure and relationships
- `queries.sql` – SQL queries for performance analysis

---

- MySQL / SQL
