# College Database Management System

## Project Overview
Designed a relational database to manage student, course, faculty, and enrollment data for a college.

## Tables Included
- Students
- Courses
- Faculty
- Enrollments
- Departments

## Key Queries
- Student enrollment details
- Course-wise student count
- Faculty workload analysis
- Department-wise performance

## SQL Concepts Used
- Primary & foreign keys
- Joins (INNER, LEFT)
- Subqueries
- Group By and Having
