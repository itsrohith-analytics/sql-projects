# Product Catalogue Database (SQL)

## Project Overview
This project represents a Product Catalogue Database designed to manage products, categories, and pricing information.  
It helps analyze product distribution, category performance, and pricing trends using SQL queries.

---

## Database Entities
- **Product** – Stores product details such as name, price, and category
- **Category** – Stores product category information

---

## Key Business Questions Answered
- List of all products with their categories
- Category-wise product count
- Average product price by category
- Identification of expensive products
- Categories with a high number of products
- Products priced above category average

---

## SQL Concepts Used
- Relational database design
- Primary and foreign keys
- INNER JOIN
- Aggregate functions (COUNT, AVG, MAX)
- GROUP BY and HAVING clauses
- Subqueries

---

## Files Included
- `schema.sql` – Database structure and relationships
- `queries.sql` – SQL queries for product and pricing analysis

---

## Tools Used
- MySQL / SQL
